# Glory Joseph Mamani - Professional Portfolio

Welcome to the repository for my personal and professional portfolio website, hosted on GitHub Pages. This site serves as a comprehensive digital resume highlighting my background in pharmacology, health education, and academic research.

## Live Site
You can view the live website here: [Glory Joseph Mamani](https://glorythe21st-web.github.io/)

## Project Overview
This repository contains the source code for a multi-page static website. The site is built using Markdown and HTML, and it is automatically generated and styled using Jekyll's native GitHub Pages integration. 

## Site Structure
The portfolio is divided into the following sections for easy navigation:
* **Home (`index.md`):** Professional summary and contact information.
* **Education (`education.md`):** Academic history and degree details.
* **Skills (`skills.md`):** Teaching, research, and technical competencies.
* **Research (`research.md`):** Undergraduate projects and scholarly contributions.
* **Experience (`experience.md`):** Professional history, internships, and community service.
* **Certifications (`certifications.md`):** Professional development and completed courses.
* **Memberships (`memberships.md`):** Professional associations and volunteer work.
* **Awards (`awards.md`):** Scholarships and academic recognition.
* **Languages (`languages.md`):** Linguistic proficiencies.
* **References (`references.md`):** Professional references.

## Technologies Used
* **GitHub Pages:** Hosting and continuous deployment.
* **Jekyll:** Static site generation and theme application.
* **Markdown:** Core content formatting.
* **HTML & CSS:** Custom interactive elements, image styling, and navigation menus.
